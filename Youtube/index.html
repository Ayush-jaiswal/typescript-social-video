Youtube video Class

List of Properties

  • Number of views
  • Number of likes
  • Number of dislikes
  • Title of the video
  • Channel Name
  • Description under the video
  • Video URL
  • Publishing Date
  • Category
  • List of comments

List of Methods

  • addToCommentList (name:string,comment:string)
  • viewCommentList()
  • getVideoURL()
  • getPublishingDate()
  • getNoOfLikes()
  • incrementLikes()
  • getNoOfDislikes()
  • incrementDislikes()
  • getNoOfViews()
  • incrementNoOfViews()
  • getVideoTitle()
  • getChannelName()
  • getCategoryOfTheVideo()
  • getDescription()

Description

  • All the properties of the video are made private so that they can't be manipulated from outside the class.
  • Most of the properties are provided with getter methods only. However some properties like NoOfViews,NoOfLikes etc which can be modified by the user are provided with setter methods.
